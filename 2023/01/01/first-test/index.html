质数

  1. 质数和合数是针对所有大于 111 的「自然数」来定义的（所有小于等于 111 的数都不是质数）。

  2. 所有小于等于 111 的整数既不是质数也不是合数。

  3. 质数的判定——试除法：

    d∣nd | nd∣n 代表的含义是 ddd 能整除 nnn。（这里的 | 代表整除）

    一个合数的约数总是成对出现的，如果 d∣nd|nd∣n，那么 (n/d)∣n(n/d)|n(n/d)∣n，因此我们判断一个数是否为质数的时候，只需要判断较小的那一个数能否整除 nnn 就行了，即只需枚举 d≤(n÷d)d\le(n\div d)d≤(n÷d)，即 d×d≤nd\times d\le nd×d≤n，d≤nd\le \sqrt{n}d≤n​ 就行了。

    sqrt(n) 这个函数执行的时候比较慢。

    i∗i<=ni * i <= ni∗i<=n 又容易溢出。

  4. 分解质因数——试除法（算数基本定理）

    算数基本定理：任何一个大于 111 的自然数 nnn，如果 nnn 不为质数，那么 nnn 可以唯一分解成有限个质数的乘积 n=P1a1P2a2P3a3…Pnann=P_1^{a1}P^{a2}_2P^{a3}_3\dots P^{an}_nn=P1a1​P2a2​P3a3​…Pnan​，这里 P1<P2<P3<⋯<PnP_1<P_2<P_3<\dots <P_nP1​<P2​<P3​<⋯<Pn​ 均为质数，其中指数 aiaiai 是正整数。

    特别要注意——分解质因数与质因数不一样，分解质因数是一个过程，而质因数是一个数。

    一个合数分解而成的质因数最多只包含一个大于 n\sqrt{n}n​ 的质因数。（反证法，若 nnn 可以被分解成两个大于 n\sqrt{n}n​ 的质因数，则这两个质因数相乘的结果大于 nnn，与事实矛盾）

    当枚举到某一个数 iii 的时候，nnn 的因子里面已经不包含 [2,i−1][2,i−1][2,i−1] 里面的数（已经被除干净了），如果 n∣in | in∣i，则 iii 的因子里面也已经不包含 [2,i−1][2,i−1][2,i−1] 里面的数，因此每次枚举的数都是质数。

    质因子（质因数）在数论里是指能整除给定正整数的质数。根据算术基本定理，不考虑排列顺序的情况下，每个正整数都能够以唯一的方式表示成它的质因数的乘积。

    两个没有共同质因子的正整数称为互质。因为 111 没有质因子，111 与任何正整数（包括 111 本身）都是互质。

    只有一个质因子的正整数为质数。

  5. 筛质数（朴素筛法）

    步骤：把 [2,n−1][2,n−1][2,n−1] 中的所有的数的倍数都标记上，最后没有被标记的数就是质数。

    原理：假定有一个数 ppp 未被 [2,p−1][2,p−1][2,p−1] 中的数标记过，那么说明，不存在 [2,p−1][2,p−1][2,p−1] 中的任何一个数的倍数是 ppp，也就是说 [2,p−1][2,p−1][2,p−1] 中不存在 ppp 的约数，因此，根据质数的定义可知：ppp 是质数。

    调和级数：当 nnn 趋近于正无穷的时候，12+13+14+15+⋯+1n=ln⁡n+c\frac{1}{2}+\frac{1}{3}+\frac{1}{4}+\frac{1}{5}+\dots+\frac{1}{n}=\ln n+c21​+31​+41​+51​+⋯+n1​=lnn+c（ccc 是欧拉常数，约等于 0.5770.5770.577 左右）

    时间复杂度：约为 O(nlog⁡n)O(n\log n)O(nlogn)（注：此处的 log⁡\loglog 特指以 222 为底）

  6. 埃氏筛（稍加优化版的筛法）

    质数定理：1∼n1\sim n1∼n 中有 n/ln nn/ln\ nn/ln n个质数。

    步骤：在朴素筛法的过程中只用质数项去筛。原理其实很简单，所有的合数都可以用质因数的乘积来表示，所以我们只需要筛掉质数的倍数，合数自然也就被筛掉了。

    时间复杂度：O(nlog⁡(log⁡n))O(n\log(\log n))O(nlog(logn))。

  7. 线性筛

    若 n≈106n\approx10^6n≈106，线性筛和埃氏筛的时间效率差不多，若 n≈107n\approx10^7n≈107，线性筛会比埃氏筛快了大概一倍。

    埃氏筛法尽管利用了质数的倍数一定不是质数来构造筛子，但还是有重复计算，如 303030 不是质数，在 i=2i = 2i=2时候，2∗152 * 152∗15 被筛掉了一次，在 i=5i = 5i=5 时候，5∗65 * 65∗6又被筛掉了一次，造成效率的浪费。

    我们在枚举倍数时，保证当前质数不能整除这个倍数就好了，当然了，在之前要先筛一次，也就是p是m的p，这样我们在最优性的条件下，对每一个质数，它所有对应的倍数，都被枚举到，也就保证了可行性。

    核心思想：所有的合数可以表示为两个或多个质数的积。

    核心：1∼n1\sim n1∼n 内的合数 ppp 只会被其最小质因子筛掉。（算数基本定理）

    原理：1∼n1\sim n1∼n 之内的任何一个合数一定会被筛掉，而且筛的时候只用最小质因子来筛，然后每一个数都只有一个最小质因子，因此每个数都只会被筛一次，因此线性筛法是线性的。

    枚举到 iii 的最小质因子的时候就会停下来，即 if(i % primes[j] == 0) break;

    当 i % primes[j] != 0 时，primes[j] 一定小于 iii 的最小质因子，primes[j] 一定是 primes[j]*i 的最小质因子。
    当 i % primes[j] == 0 时，primes[j] 一定是 iii 的最小质因子，而 primes[j] 又是 primes[j] 的最小质因子，因此 primes[j] 是 primes[j]*i 的最小质因子。

    里面关键就是那个 breakbreakbreak，如果不 breakbreakbreak，就是再往大里枚举，那么以后枚举到的质数，一定不是最小质数，因为它都比当前质数大了，怎么可能最小，最小也只可能是当前的质数，从而实现了优化。

完全平方数

  1. 一个整数 aaa 是一个完全平方数，是指它是某一个整数的平方，即存在一个整数 bbb ，使得 a=b2a=b^2a=b2
  2. 一个正整数如果是完全平方数，那么它的质因数一定全部都是是偶数次。
    例如 36=22∗3236 = 2^2 * 3^236=22∗32 $ 900 = 2^2 * 3^2 * 5^2$
  3. 一个非完全平方数则至少有一个是奇数次的质因数。
    例如 18=21∗3218 = 2^1 * 3^218=21∗32
    若使 181818 变为完全平方数，就要使得奇数次变为偶数次，这里就可以乘以一个 222 使得 212^121 变为 222^222，最后得到的 18∗2=3618 * 2 = 3618∗2=36 就是我们需要的完全平方数。
